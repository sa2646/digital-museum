# Carnival in Rio: Performance, Power & Identity

A digital museum interactive exhibit exploring the Brazilian Carnival in Rio de Janeiro, focusing on samba school parades at the Sambadrome.

## 🎭 About This Project

This website presents an immersive digital museum experience that examines the cultural significance of Rio's Carnival through:
- Historical timeline and traditions
- Samba school preparations and competitions
- The spectacular Sambadrome parades
- Cultural identity and social commentary

## ✨ Features

### Navigation
- Sticky navigation bar with smooth scroll effects
- Responsive hamburger menu for mobile devices
- Hover animations and active states

### Hero Section
- Full-screen immersive design with gradient overlay
- Dramatic typography with Playfair Display font
- Gold call-to-action button with hover effects
- Parallax scrolling effect

### Exhibit Cards
- Four themed cards (History, Samba Schools, The Parade, Politics & Identity)
- Image zoom on hover
- Gradient overlays for text readability
- Responsive grid layout

### Timeline Section
- "Carnival Through Time" with four historical periods
- Scroll-triggered animations
- Elegant dark theme with gold accents

### Interactive Gallery
- Responsive image grid
- Click-to-open lightbox viewer
- Hover zoom effects
- Keyboard navigation (ESC to close)

### Samba Rhythm Section
- Audio player with custom controls
- Animated play/pause button
- Visual feedback when playing

## 🎨 Design System

- **Colors**: Deep navy background (#0a1929), gold accents (#e3b04b), emerald green highlights
- **Typography**: Playfair Display for headings, Inter for body text
- **Animations**: Smooth scroll reveals, parallax effects, hover states
- **Responsive**: Fully optimized for desktop, tablet, and mobile

## 🚀 Deployment

This site is designed for GitHub Pages deployment. To deploy:

1. Fork this repository
2. Enable GitHub Pages in repository settings
3. Select source as "Deploy from a branch" and choose "main"
4. Your site will be available at `https://[username].github.io/carnival-museum/`
