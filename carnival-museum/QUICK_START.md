# Carnival Museum

## 🎭 Digital Museum Exhibit
**Carnival in Rio: Performance, Power & Identity**

An interactive digital museum experience exploring Brazilian Carnival culture, samba school parades, and cultural identity.

## 🚀 Quick Start

### View Live Demo
Open `index.html` in your browser or visit the GitHub Pages deployment.

### Local Development
Simply open `index.html` in your web browser - no server required!

### GitHub Pages Deployment
1. Push to GitHub repository
2. Enable GitHub Pages in settings
3. Select source branch (main)
4. Site will be live at `https://[username].github.io/carnival-museum/`

## 📁 Files Overview
- `index.html` - Main website structure
- `styles.css` - Complete styling and animations
- `script.js` - Interactive features
- `images/` - Carnival photography (placeholders included)
- `audio/` - Samba rhythm audio (placeholder included)

## ✨ Key Features
- 🎨 Museum-quality design with Carnival aesthetics
- 📱 Fully responsive for all devices
- 🎭 Interactive gallery with lightbox
- 🎵 Samba rhythm audio player
- ⏱️ Historical timeline
- 🖼️ Exhibit cards with hover effects
- 📜 Smooth scroll animations

## 🎨 Design
- **Colors**: Navy background, gold accents, emerald highlights
- **Fonts**: Playfair Display (headings), Inter (body)
- **Animations**: Scroll reveals, parallax, hover states

Perfect for educational institutions, cultural organizations, or anyone interested in Brazilian Carnival heritage!
